# 🐾 Zack's Pet Store

Welcome to **Zack's Pet Store**, a responsive web project where users can browse and adopt pets such as dogs, cats, birds, and even capybaras! Built with simple HTML, CSS, and JavaScript, this website is ideal for learning or demonstrating beginner-level front-end development.
